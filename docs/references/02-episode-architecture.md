# 分集架构系统

## 输出

1. `分集目录.md`
2. `项目进度.md`

## 单集骨架

1. 开场钩子（0-3秒）
2. 冲突升级（3-45秒）
3. 局部反转（45-60秒）
4. 集尾悬念（最后5秒）

## 情绪曲线规则

1. 连续3集强度不得低于5/10
2. 压抑段不超过3集，必须释放
3. 免费段末（18-21集）必须有强付费卡点

## 分集字段最小集

| 集数 | 标题 | 完整事件流程 | 内容概括 | 核心冲突 | 爽点 | 反转 | 钩子 | 情绪类型 | 强度 |
|---|---|---|---|---|---|---|---|---|---|

**字段说明**：
- **完整事件流程**：用箭头标注当集的完整事件流程，如"宣布 → 搜身 → 签字 → 离开"
- **内容概括**：用一两句话概括当集内容

## 事件完整性原则

### 核心原则
1. **一个完整事件应该在一集内完成**
2. **不要把连续动作拆成多集**
3. **每集应该是一个独立的故事单元**
4. **相邻集之间应该是不同的场景或事件**

### 完整事件的定义
一个完整事件包含：
- 明确的开始（触发点）
- 中间的发展（冲突/行动）
- 明确的结束（结果/转折）

### 反例：事件拆分错误
❌ **错误示例**：
- 第1集：被赶出萧家（搜身 + 签字 + 离开）
- 第2集：被赶出萧家（搜身 + 签字 + 离开）← 重复内容

**问题**：把同一个事件重复了两次，没有理解"完整事件"的概念

### 正例：事件完整且不重复
✅ **正确示例**：
- 第1集：被赶出萧家
  - 完整事件流程：宣布驱逐 → 搜身 → 揭露真相 → 签字 → 纪念册 → 离开
  - 内容概括：萧家宣布驱逐，搜身揭露真相，主角签字离开
- 第2集：拍卖会展示实力
  - 完整事件流程：入场 → 竞拍 → 展示实力 → 震惊全场
  - 内容概括：主角在拍卖会上展示实力，震惊全场

### 如何判断事件是否完整
检查清单：
- [ ] 事件有明确的开始和结束
- [ ] 事件的所有关键步骤都在当集完成
- [ ] 下一集是新的场景或新的事件
- [ ] 没有把连续动作（如搜身、签字、离开）拆成多集

## 判定标准

1. PASS：字段完整，曲线可执行，卡点明确，**每集包含完整事件流程和内容概括，相邻集无重复**
2. FAIL：缺反转/钩子、强度平直、进度表不可追踪、**事件拆分不合理、相邻集有重复内容**

## 三幕式结构理论基础

### 核心概念

三幕式结构是经典的剧本结构理论，适用于全剧整体架构和单集剧情设计。

**核心机制**：触发 → 冲突 → 解决

**篇幅比例**：
- 第一幕（建立）：25%
- 第二幕（对抗）：50%
- 第三幕（解决）：25%

### 全剧架构应用（80集短剧）

#### 第一幕：建立（第1-20集，25%）

**核心任务**：
1. **介绍背景**：世界观、时代背景、故事调性
2. **明确主角**：主角的身份、性格、目标、可爱之处
3. **预示冲突**：暗示即将到来的冲突和困境
4. **提供悬念**：抛出让人想知道答案的问题

**关键节点**：
- 第1集：开场钩子，建立世界观
- 第10集：一卡付费点，主角被召唤踏上旅程
- 第20集：第一幕结束，主角正式进入冲突

#### 第二幕：对抗（第21-60集，50%）

**核心任务**：
1. **面对冲突**：主角面对大大小小、接连不断的冲突
2. **遭遇失败**：主角必须遭遇失败，从低谷中重新崛起
3. **成长改变**：主角在磨难中逐步蜕变成熟
4. **节节升高**：戏剧张力节节升高，环环相扣

**关键节点**：
- 第30集：二卡付费点，主角遭遇重大挫折
- 第40集：中点转折，主角开始反击
- 第50集：冲突升级，主角面临最大困境
- 第60集：三卡付费点，主角准备最终对决

#### 第三幕：解决（第61-80集，25%）

**核心任务**：
1. **解决冲突**：主角解决最后的冲突，达成或未达成目标
2. **揭开伏笔**：所有遗留未解的谜团都在此解开
3. **震撼结局**：准备一个震撼人心的真相或结局

**关键节点**：
- 第70集：最终对决开始
- 第80集：大结局，所有伏笔揭开

### 单集结构应用（1-2分钟单集）

#### 第一幕：触发（前30秒，25%）

**核心任务**：
- 开场钩子（0-3秒）：抓住观众注意力
- 触发事件（3-30秒）：引发本集的核心冲突

#### 第二幕：冲突（中间60秒，50%）

**核心任务**：
- 冲突升级（30-90秒）：主角应对冲突，遭遇困难
- 局部反转（90-120秒）：情节出现转折

#### 第三幕：解决（后30秒，25%）

**核心任务**：
- 解决冲突（120-145秒）：本集冲突得到解决
- 集尾悬念（145-150秒）：留下钩子，引向下一集

### 系列剧结构应用

**双层结构**：
- **单集层面**：每一集都符合三幕式结构（触发 → 冲突 → 解决）
- **全剧层面**：全系列也符合三幕式结构（建立 → 对抗 → 解决）

**示例**：
- 第1集：触发（被赶出萧家）→ 冲突（搜身揭露）→ 解决（签字离开）→ 钩子（神秘电话）
- 第2集：触发（接到邀请）→ 冲突（拍卖竞争）→ 解决（展示实力）→ 钩子（身份曝光）
- ...
- 全剧：建立（1-20集）→ 对抗（21-60集）→ 解决（61-80集）

### 使用建议

1. **全剧规划优先**：先用三幕式结构规划全剧的整体架构
2. **单集设计跟随**：再用三幕式结构设计每一集的剧情结构
3. **灵活调整比例**：25%-50%-25%不是僵硬的铁则，可根据实际情况调整
4. **确保任务完成**：每一幕的核心任务必须完成，比例可以灵活

**详细理论参考**：`sources/script-structure-methods/影剧本经典结构三幕式结构通俗详解.txt`

## 10集定生死：前10集标准化流程

### 核心认知

90%的毙稿，都死在前10集。前10集不是在"讲故事"，是在"建危楼"——用最快的速度，建一栋看似摇摇欲坠、实则结构精准的"情绪危楼"，把观众和主角一起"困"在里面。

**10集生死线**：
- **1-3集**：打下背叛的地基，焊死逆袭的钢筋
- **4-7集**：砌好职场的墙，只留一扇叫"真相"的窗
- **8-10集**：点燃炸药，炸掉第一面墙，让观众看到外面还有更大的阴谋

**目标**：第10集结束时，观众必须"累并爽着"，且清楚地知道，真正的商战才刚刚开始。

### 1-3集：背叛、底牌、软肋三件套

这三集是"产品说明书"，必须让所有人（观众、资方、算法）看懂你要卖什么。

#### 第1集·黄金模版（核心：极致的辱+隐秘的伏笔）

- **开场（0-30秒）**：用最直观的画面，展示主角的"社会性死亡"
- **中段（31-90秒）**：羞辱升级，并伤及他在乎的人（**软肋出现**）
- **结尾（91-120秒）**：绝境中，底牌"预启动"（**外挂伏笔**）

#### 第2集·绑定模版（核心：底牌到账+首次小试牛刀）

**任务**：正式激活底牌，并让主角用它解决一个"与主线相关"的小问题。

#### 第3集·期待模版（核心：小胜后的更大危机+主线反派露牙）

**任务**：给一次"有限的胜利"，然后立刻用"更大的阴影"覆盖它。

至此，观众拥有：一个想守护的人（软肋），一个想踩碎的反派，一个期待的武器（底牌）。**追剧铁三角搭建完毕**。

### 4-7集：压力测试与盟友陷阱

这三集是"压力泵"，任务是不断加压，并引入变量。

#### 第4-5集·行业压力集

**公式**：让主角的"社会身份"与"隐藏能力"产生剧烈冲突。

#### 第6-7集·情感变量集

**公式**：引入"不可靠的盟友"或"亦敌亦友的红颜"。

**作用**：增加剧情复杂度，考验主角心性，为后期反转铺垫。

### 8-10集：首轮爆破与地图展开

这三集是"爆破点"，必须炸掉前期积累的大部分矛盾，同时炸出一个新世界。

#### 第8集·终极备战集

**任务**：所有矛盾线收束，主角明确最终目标，完成逆袭前最后准备。

#### 第9集·首轮核爆集

**任务**：在一个极致公开的场合，完成对"中期BOSS"的彻底打脸。

**节奏**：打脸过程要"有步骤、有层次、有反转"，让反派体验从天堂到地狱的完整过程。

#### 第10集·废墟新程集（最重要！）

**任务**：完成爆破，并立刻展现"废墟后的新战场"。

**核心**：让观众清晰感受到，刚刚爽完的，只是新手村。一个更庞大、更黑暗的商业世界观就此拉开序幕。

### 前10集防崩盘三条军规

1. **反派不能"只吵不咬"**：3集内，反派必须造成实质性、难以挽回的伤害（如：毁掉主角的事业、剥夺他在乎的人的希望）。言语羞辱是点心，实质伤害才是主菜。
2. **底牌不能"只亮不用"**：第2集激活，第3集就必须有一次成功的小应用，且这次应用要直接关联主线困境。
3. **节奏不能"只压不爆"**：每3集必须有一个中等以上的情绪释放点（小胜利、小反转）。连续压抑超过3集，观众会绝望弃剧。

### 10集过稿自检清单

交稿前，逐项打勾：

- [ ] 第1集，主角的"职业标签"和"核心耻辱"是否清晰无比？
- [ ] 第1集，是否出现了"危及主角软肋"的具体威胁？
- [ ] 第2集，底牌是否正式激活并完成一次有效使用？
- [ ] 第3集结尾，"中期目标BOSS"是否已浮出水面？
- [ ] 第4-7集，是否引入了至少1个有魅力的新角色（盟友或敌友）？
- [ ] 第9集，打脸场面是否发生在"公开场合"，并有"三重以上反转"？
- [ ] 第10集结尾，是否彻底解决了第一个大矛盾，同时开启了更宏大的新矛盾？

## 叙事结构类型

### 一、线性叙事

#### 1. 顺叙
按照时间/事情的发展顺序，最常规、最常见。

- **优点**：容易讲清楚，观众基本无理解成本，自媒体博主最常用的一种讲述手法
- **缺点**：容易无聊，变成流水账

#### 2. 倒叙
把最有冲突、有看点的场面放在开始，逐步倒回之前的时间点。

- **优点**：开场吸引人，引发好奇继续往下看
- **例**：不可撤销、泰坦尼克号

#### 3. 插叙
在顺叙/倒叙的基础上，加入回忆片段展开，可以让多条时间线同步进行。

- **优点**：丰富人物背景，使故事更丰满可信
- **例**：贫民窟的百万富翁、千年女优

### 二、非线性叙事

打破常规的时间及逻辑性，使观众产生沉浸感，并激发思考乐趣。

#### 1. 多线叙事
从故事架构层面打破传统线性时间顺序，多条故事线以非顺序方式呈现。这些故事线可以在不同时空展开，相互之间可能存在因果、呼应等复杂关系，最终交汇融合，为观众呈现一个完整且独特的故事全貌。

- **优点**：群像塑造、错乱时空的故事线交织形成的命运感、观众主动联系因果沉浸感强
- **例**：低俗小说、穆赫兰道

#### 2. 环形叙事
循环，通常在无限流题材上有所表现。

- **优点**：天然的悬念钩子；深度的沉浸感；宿命感
- **例**：土拨鼠之日、忌日快乐、恐怖游轮

#### 3. 碎片化叙事
将故事的情节打碎，按照特定的顺序重新组合，让观众在观影过程中拼凑出完整的故事。

- **优点**：沉浸感、探索性、不同的解读
- **例**：记忆碎片

### 三、提取重点

常见：顺叙+插叙的结构——讲解+空镜、回忆

| 提纯 | 综艺真人秀：24小时观察录制提取重点性事件 | vlog：从无趣的生活片段中提取有趣的事 |
|------|--------------------------------------------|--------------------------------------|
| 融合 | 预告宣传：把2个小时浓缩到1min              | 混剪：把几十部作品融合到一起          |
| 前置 | 黄金2s/3s/5s                              |                                      |

## 叙事视角选择

### 1. 第一人称
透过主角的眼睛看世界，大量主角心理旁白。

- **特点**：主角不知道的你也不知道；有很强的代入感
- **例**：美国丽人、搏击俱乐部

### 2. 第三人称

| 视角类型 | 核心特点 | 电影示例 |
|----------|----------|----------|
| **全知** | 上帝视角 | 泰坦尼克号、寄生虫 |
| **限制** | 跟随角色 | 看不见的客人 |
| **多重** | 碎片悬疑 | 低俗小说 |
| **客观** | 克制冷静 | 贾科长 |

### 使用建议

1. **短剧推荐**：第三人称全知视角，便于快速推进剧情和展示多角色冲突
2. **悬疑题材**：可使用限制视角或多重视角，增强悬念感
3. **情感题材**：可适度使用第一人称或限制视角，增强代入感
4. **避免混乱**：单集内保持视角一致，避免频繁切换导致观众出戏
