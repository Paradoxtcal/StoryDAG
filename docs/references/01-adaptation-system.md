# 小说改编分析系统

## 输入

1. 原文章节或剧情梗概
2. 题材目标（短剧/动态漫）

## 输出

1. `cleaned_text.txt`
2. `分析报告.md`
3. `角色档案.md`

## 核心分析维度

1. **题材判定**：男频/女频，核心爽点类型（参见：男女频底层逻辑）
2. **冲突引擎**：主冲突、次冲突、每集可用冲突池
3. **爽点引擎**：打脸、反转、关系变化、能力兑现（参见：三层Buff叠加）
4. **角色系统**：目标/障碍/关系/弧光
5. **一致性锚点**：称呼规范、关键道具、时间线、世界规则
6. **情绪曲线**：全剧情绪强度规划，关键节点设计（参见：情绪曲线管理）

## 称呼规范最低模板

| 角色 | 对象 | 正确称呼 | 禁用称呼 |
|---|---|---|---|
| 母亲 | 儿子 | 儿子/小X | 弟弟 |

## 判定标准

1. PASS：前10集冲突与爽点可追溯，角色锚点完整，男女频判定明确，情绪曲线规划完整
2. FAIL：缺称呼规范表、冲突空泛、角色目标不清、男女频判定缺失

## 男女频底层逻辑

### 核心判断（必须执行）

分析小说后，首先判断题材属性：

**女频特征：**
- 核心爽点：情绪爽（关系变化）
- 观众期待：要看"他怎么对我"
- 高光时刻：被宠、被护、被选择、被偏爱
- 常见桥段：误会拉扯、保护心动、背叛后悔、追妻火葬场

**男频特征：**
- 核心爽点：能力爽（地位翻盘）
- 观众期待：要看"我怎么变强"
- 高光时刻：技能炫技、谈判反杀、实力碾压
- 常见桥段：被羞辱→打脸、被压制→绝地反击、全场震惊

### 四维度对比

| 维度 | 男频 | 女频 |
|------|------|------|
| 目标导向 | 能力提升、地位逆袭 | 情感满足、关系改善 |
| 冲突类型 | 实力碾压、打脸羞辱 | 情感纠葛、误会虐心 |
| 情绪需求 | 爽感（我很强） | 甜感/虐感（被爱/被伤害） |
| 节奏偏好 | 快节奏、高密度打脸 | 情绪起伏、拉扯暧昧 |

### 镜头语言差异

**女频镜头语言：**
- 多特写、多眼神戏、多情绪停顿
- 小动作特写（牵手、递伞、对视）
- 观众吃的是"情绪氛围感"

**男频镜头语言：**
- 多中远景、多动作展示、多场面调度
- 技能表现和结果
- 观众吃的是"能力冲击感"

### 女频五大高命中率情绪组合

1. **背叛 + 后悔 + 追妻火葬场**（命中率最高）
2. **倒计时 + 离开/死亡/出国**（制造不可逆性）
3. **重新选择 + 不原谅 + 男二上位**（情绪反击）
4. **协议婚姻/先婚后爱 + 总裁**（低解释成本）
5. **绿茶/女兄弟/白月光**（稳定冲突制造器）

## 故事导向类型（5个保命公式）

从选题到故事构建，需要明确故事的核心导向。导向是指故事中最出彩的元素，是吸引观众持续观看的粘性来源。

### 五种导向类型

#### 1. 人物导向型

**定义**：以人物为重点，人物本身很牛逼。

**特征**：
- 看点来自"他是一个什么样的人、身份"
- 人物身份本身就是吸引力（龙王、神医、女总裁等）
- 参考：《教父》《阿甘正传》

**短剧应用**：
- 主角身份标签极致化（战神、龙王、隐藏大佬）
- 身份揭晓是核心爽点
- 适合掉马甲类型剧本

#### 2. 目标导向型

**定义**：故事的目标是什么？

**特征**：
- 不管人物身份，考虑"他要做的事情是什么"
- 目标本身足以拉起观众兴趣
- 常见目标：复仇、赚钱、追妻、盗墓

**短剧应用**：
- 目标必须明确且贯穿全剧
- 每集推进目标进度
- 适合复仇、逆袭类型剧本

#### 3. 困境导向型

**定义**：主角面临的困境本身就是看点。

**特征**：
- 看点在于世界观设定（末日、破产、被冤入狱、被追杀）
- 困境制造压迫感
- 在短剧中应用较少（因为短剧核心是爽）

**短剧应用**：
- 谨慎使用，避免过度压抑
- 可用于末日题材
- 必须快速转入逆袭

#### 4. 奇遇导向型

**定义**：普通人接触不到的超现实元素。

**特征**：
- 穿越、重生、金手指、大佬传功
- 吸引点在于"化学反应"和"命运改变"
- 观众期待：穿越后会怎样？重生能否改变命运？

**短剧应用**：
- 高命中率类型
- 重生、穿越、系统流
- 金手指使用是核心看点

#### 5. 谜题导向型

**定义**：故事核心是解谜。

**特征**：
- 需要观众思考
- 与短剧"无脑爽"属性相冲突
- **不建议新手使用**

**短剧应用**：
- 极度谨慎
- 需要极高的编剧功力
- 需要平台敢拍

### 导向类型选择原则

1. **可叠加使用**：一个故事可以同时包含多种导向（如：人物导向 + 奇遇导向 = 重生战神）
2. **必有主导向**：确定最出彩的元素作为主导向
3. **服务于爽点**：所有导向最终服务于"解压"和"代入感"
4. **粘性来源**：导向类型决定了观众为什么要看完100集

### 导向与题材匹配

| 导向类型 | 适合题材 | 核心吸引力 |
|---------|---------|-----------|
| 人物导向 | 战神、龙王、隐藏大佬 | 身份揭晓 |
| 目标导向 | 复仇、逆袭、追妻 | 目标达成 |
| 困境导向 | 末日、绝境求生 | 困境突破 |
| 奇遇导向 | 重生、穿越、系统 | 能力兑现 |
| 谜题导向 | 悬疑、推理 | 真相揭晓 |

**重要提示**：在设计大纲之前，必须明确故事的主导向类型，这决定了整个故事的叙事重心和节奏分配。

## 三层Buff叠加公式

**套路梗（冲突核心）→ 元素梗（节奏高潮）→ 人设梗（引流维持）**

### 一、套路梗：冲突矛盾的核心

套路梗是短篇爽文里绝对的主角，整篇文章的冲突矛盾都是围绕它展开的。

**典型示例：兼祧两房**
- 男主背着"继承遗志"理由出轨女二
- 明面上人设都很好，只有女主知道真相
- 旁人反而觉得女主不懂情理
- 虐心反转，又气人又精彩

### 二、元素梗：节奏和冲突高潮

元素梗是给文章打额外buff的工具，用来制造节奏和冲突高潮。

**典型示例：直播梗**
- 反派把主角丑态直播出来，引来网暴
- 主角精神崩溃又无助
- 虐得深，拉满互动量

### 三、人设梗：引流和维持关注

人设梗是锦上添花的关键，用极端或鲜明的人物设定吸引读者。

**作用：**
- 提高点开率
- 引流维持关注
- 产生路径依赖（人们喜欢熟悉设定）

### 三层Buff叠加的本质

- 套路梗：提供情绪价值（虐/爽）
- 元素梗：提供节奏价值（高潮）
- 人设梗：提供流量价值（吸引）

**记住：不是buff越多越好，是buff配合得越好越好。**

## 情绪曲线管理

### 核心理念

短剧的本质是「情绪过山车」。观众不是在看故事，是在体验情绪。

情绪曲线设计的铁律：**压力积累 → 爽点释放 → 短暂平息 → 更大压力 → 更强爽点**，螺旋上升，直到大结局。

### 全剧情绪曲线模板（60集参考）

```
情绪强度
  ▲
10│                                    🔥🔥🔥终极高潮
  │                              🔥高潮
  │                        🔥
  │              🔥              ↘至暗
  │        🔥         ↘低谷
  │   🔥        ↘低谷
  │🔥
  │ ↘平息  ↘平息
  └────────────────────────────────▶ 集数
   1   10  20  30  40  50  60
       💰付费卡点
```

### 四阶段强度分配

| 阶段 | 集数 | 强度基线 | 节奏特征 | 核心任务 |
|------|------|---------|---------|---------|
| 引爆期 | 1-15% | 中高 | 密集冲突 | 建立人设、初始爽点、吸引观众 |
| 蓄力期 | 15-40% | 中 → 高 | 逐步升温 | 扩大冲突、引入新势力、蓄力 |
| 爆发期 | 40-80% | 高 → 极高 | 高密度反转 | 多线汇聚、连续爽点、大逆袭 |
| 收束期 | 80-100% | 极高 → 收束 | 先快后稳 | 终极对决、情感升华、大结局 |

### 关键节点设计

| 节点 | 位置（占总集数比） | 内容要求 |
|------|----------------|---------|
| 爆炸开场 | 第1集 | 30秒内引爆核心矛盾，奠定基调 |
| 初始爽点 | 约5%处 | 主角第一次小胜利，留住观众 |
| 💰付费卡点 | 约15-20%处 | 最强悬念，身份/能力即将揭晓但被截断 |
| 第一转折 | 约25%处 | 主角处境发生根本性变化 |
| 新势力登场 | 约35%处 | 引入更强对手或更复杂局面 |
| 中点大反转 | 约50%处 | 故事走向重大转折，观众预期被打破 |
| 至暗时刻 | 约65%处 | 主角遭遇最大危机，一切似乎失去 |
| 终极反击启动 | 约75%处 | 绝地反击，进入终局模式 |
| 高潮连爆 | 约85-90%处 | 多条线索汇聚，连续爆发 |
| 大结局 | 最后3-5集 | 终极对决 + 情感收束 + 圆满/反转结局 |

### 每集情绪类型标注

| 情绪类型 | 标记 | 特征 | 使用频率 |
|---------|------|------|---------|
| 爽点爆发 | 💥 | 强反转、打脸、逆袭 | 主力，约40%的集 |
| 压力蓄积 | 🔨 | 主角被压制、反派嚣张 | 约25%，为爽点铺垫 |
| 情感波动 | 💔 | 误解、心动、决裂 | 约20%，调节节奏 |
| 悬念引爆 | ❓ | 真相将近、线索汇聚 | 约10%，在关键节点 |
| 平缓过渡 | ➡️ | 信息交代、关系推进 | 约5%，尽量压缩 |

**核心原则：连续3集不能出现相同情绪类型标记。平缓过渡集（➡️）绝不能连续出现。**

### 情绪强度警戒规则

- 连续2集强度 ≤ 4：提示「节奏过缓，需检查」
- 连续2集强度 = 10：提示「情绪透支，建议在中间插入平息集」

### 人设设计原则

1. **现实人物常混合多种特质**（如"狡猾+老练"、"天真+乐观"），需灵活叠加句式
2. **设置性格锚点**，从细节处增加真实感和记忆点，比如为角色设计标志性句式或口头禅（如自恋型总说"凡人啊~"，观型总说"没问题!"，胆小型说话卡顿）
3. **同一人物在不同场景也可能激发不同性格打造反差**，增加层次感（如冷漠型对陌生人惜字如金，对宠物却话痨；温柔的人被逼到绝境也会变得暴虐）
4. **同一人物不同时期语言也有所不同**，人物具备成长弧线，性格动态发展
5. **不同性格放在一起碰撞**，增加故事冲突，戏剧张力（比如直接的人VS弯弯绕绕的人；理想主义VS现实主义；暴躁VS温乘；乐观VS悲观；时间囚徒长上道，贵族vs市井等等）
6. **人是复杂的，性格是多样的**。面对不同的人，不同的场景，可能又有不一样的表现，但变化中会有一个主性格。不要太单一或是多变显得割裂。1+1，或是1+2
